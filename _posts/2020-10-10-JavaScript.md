## 类型
 - console.log() 打印结果的函数 (!:console,log() 一定会有执行效果,但不一定会有执行结果)
 - if .. else循环 [如果...否则] (根据if判断的条件语句来执行) (if 判断当条件无法进行第一次判断时,会通过第二次布尔值判断来执行)
     - 关键字:break/contiun
     - 运算符 %(取余) /(除号符) +/`+=`  -  =(赋值符) ==(等于符)
     - Eg:`var a = 0; if (a = 10) {}`

 - for 循环 (for循环的条件语句中第二条语句如果为`<`,则表示循环的总次数;for循环的执行顺序 "12体 32体 32体...)
